Timewise IDO

IMPORT DATA and display it in OUTLOOK

1. IMPORT

Import Data from Timewise Export Module - use
"Lokal Venue Appointment With Venue".

2. DATA

Go to SETTINGS and create exports. Paste the export url-string from Timewise. Then manipulate the imported data with filters for date and venue.

This software allows an unlimited amount of exports with different filters.

3. OUTLOOK

Once you've created the export, simply press the button to download the ical-export and add it into your Outlook.

Outlook will now fetch data from Timewise IDO - the data from IDO will be updated towards Timewise for each fetch.

The Ical export can be used with every calendar software allowing ical imports.
